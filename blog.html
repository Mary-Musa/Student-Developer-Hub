My First Week at Mctaba Labs

My first week at Mctaba Labs was about mastering the structure and organization of how web apps work.
I learned how to structure multi-page websites, organize files logically, and use semantic HTML5 for accessibility and maintainability.
It motivated me to focus on clean and professional web development practices from day one.

Why Semantic HTML Matters

Semantic HTML gives meaning to the content. Tags like <header>, <article>, <section>, and <footer> improve readability, accessibility, and search engine optimization.

Applying semantic structure to my projects, like the "Student Developer Hub" portfolio, made my code cleaner and easier to maintain.
It also improved user experience for visitors who are visiting the website.

The Project I'm Most Proud Of

My proudest project is the "Student Developer Hub" portfolio v2.
It's a multi-page website with sections for projects, blogs, and contact information.
I focused on structured layout, clear navigation, and deep-dive pages for each project.

This project taught me the importance of planning, organizing content, and presenting work in a professional and user-friendly way.
