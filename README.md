# Student Developer Hub | Portfolio v2

## Overview
The Student Developer Hub is a professional multi-page web portfolio built by Mary Musa. This project serves as a digital identity showcasing web development skills, projects, blog posts, and contact functionality.  

It demonstrates clean HTML5 structure, semantic markup, navigation linking, and interactive features like forms, media embedding, and FAQs.

## Features

### Core Pages
1.Home (index.html)  
   - Elevator pitch with introduction and purpose  
   - Article cards linking to main sections (About, Projects, Blog, Contact)

2.About (about.html) 
   - High-quality profile image  
   - Journey timeline and background  
   - Professional and educational highlights  

3.Projects (projects.html) 
   - List of at least 6 web applications and systems  
   - Links to deep-dive pages for top 3 projects  
   - Each project explains its purpose, approach, and key learnings  

4.Project Deep Dives (project-1.html, project-2.html, project-3.html) 
   - Detailed explanation of:
     - Problem the project solves  
     - Approach and implementation  
     - Challenges and lessons learned  

5. Blog (blog.html)
   - Table of contents linking to individual posts  
   - Minimum 3 posts describing learning journey and development insights  
   - Embedded media: images, videos, audio, and external YouTube content  

6. Contact (contact.html) 
   - Full contact form: name, email, subject dropdown, radio buttons for preference, and message textarea  
   - FAQ section using `<details>` and `<summary>` for interactive answers  


## Technologies Used
- HTML5 – Semantic structure and multi-page layout  
- CSS3 (optional) – For styling and layout improvements (can be added later)  
- Media Embeds – Images, video, audio, and YouTube iframe  
- Fully functional **form elements** with labels, dropdowns, radio buttons, and textarea
